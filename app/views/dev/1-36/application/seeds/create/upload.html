Upload a file

Download the csv template.

Open the file in whatever software program you use to edit comma-separated values (CSV) files. Fill in the template using the guidance notes on how to edit the file, then save and upload it.

Guidance on how to edit the file

  1. You must complete all fields that are mandatory.
  2. If you do not know the botanical name, or your product is made of many species, enter a description instead.
  3. You must enter a lot reference. You can enter a growing season inspection (GSI) reference.
  4. For quantity units and chemically treated you must choose from the options listed and delete any options you do not use.
  5. You must use the correct country code for the country where the seed was grown and the destination countries. You must separate these only with spaces, do not use commas. You can find the 2 letter country code at https://www.gov.uk/government/publications/uk-trade-tariff-country-and-currency-codes/uk-trade-tariff-country-and-currency-codes (opens in new tab). The only exceptions are if the country of origin is one of these countries, in which case you must enter it as it appears in this list: Aland Islands, Alaska, Azores, Bolivia, Bonaire, British Virgin Islands, Myanmar, Canary Islands, Cape Verde, Cocos Islands, Cote d'Ivoire, Curacao, Falkland Islands, Hawaii, Republic of Ireland, Kosovo, Madeira, Netherlands Antilles, Northern Ireland - use GB-NIR, Palestine - use PS, Reunion, Ross dependency, Saba, Sabah, St Barthelemy, St Eustatius, Taiwan, Wallis and Futuna Islands.
  6. Do not enter more than 500 rows of data.
  7. Save your completed file as a .CSV file and upload it.
Further advice and example template

If you are having problems with the upload

  • make sure that you have used the correct country code to enter countries
  • make sure you have not changed or deleted any columns or headers from the template
  • make sure the headers are consistently capitalised
  • check that you have not added extra commas
  • if you have edited the template using the Numbers application, be aware that this program sometimes adds extra columns to the file. If there are any empty columns without headers in your file, you should delete these columns completely.
  • check that when you edited the template, your software didn't add extra commas into your CSV file
  • for quantity units and chemically treated, make sure you have used only the options provided without changing them
  • for quantity units and chemically treated, make sure that you have deleted any options you have not used

You may find it helpful to compare your file against this example template.
