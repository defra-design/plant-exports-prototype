What you need to do

  1. Label your samples using the APHA sample references above.
  2. Send your labelled samples to the address above. You must send in any relevant copies of your import permits with your samples.
  3. Check your application status has changed to ‘report issued’. This is usually within 7 days. The application will show the sample results.
  4. Apply for phytosanitary certificates for this commodity using these sample results.
  5. You can apply for more phytosanitary certificates using the same sample. You can apply for the amount you have in stock for this batch of the commodity. If you receive more of this commodity you need to send in samples for it.

We'll only contact you if there is a problem with your application.
