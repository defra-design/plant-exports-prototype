{% if query.change == 'yes' %} If you have a large amount of a commodity and plan to export it using multiple phytosanitary certificates, you can apply for one time testing. {% else %} {% if data.build == "ux" %} If you have a large amount of a commodity and plan to export it using multiple phytosanitary certificates, you can apply for one time testing.

One time testing means you can make multiple phytosanitary applications without having to test repeatedly. The commodity you test must already be in your possession. You can re-use the sample test results to apply for phytosanitary certificates until the commodity in your possession is used up.

How it works

Step 1. Set up one time testing and send off your samples to the plant clinic. Wait for your test results to come back.

Step 2. Once your test results come back, you can start making phytosanitary certificate applications.

You must wait for your results to come back from the plant clinic before you can apply for a phytosanitary certificate.
